# BRREG Raycast Extension

En Raycast extension som lar deg søke i Brønnøysundregistrene direkte fra Raycast, uten å måtte åpne nettleseren.

## Funksjonalitet

- Søk etter bedrifter på organisasjonsnummer eller navn
- Vis grunnleggende bedriftsinformasjon
- Kopier organisasjonsnummer til utklippstavle
- Åpne bedriftsside i BRREG i nettleseren

## Teknisk implementasjon

- **API**: BRREG API v1
- **Språk**: TypeScript
- **Raycast SDK**: v1.50+
- **HTTP Client**: Fetch API

## Hvorfor vi bygde det

Som konsulenter må vi ofte slå opp bedriftsinformasjon. Denne extensionen sparer oss tid ved å:

- Unngå å åpne nettleseren
- Raskt få tilgang til organisasjonsnummer
- Effektivisere due diligence-prosesser

## Bruksområder

- Due diligence på potensielle kunder
- Verifisering av bedriftsinformasjon
- Rask tilgang til organisasjonsnummer
- Integrasjon med andre verktøy

## Fremtidige forbedringer

- Cache for bedre ytelse
- Støtte for flere søkekriterier
- Integration med andre registre
- Bulk-søk funksjonalitet
