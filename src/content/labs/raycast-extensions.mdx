# Raycast Extensions

Vi bruker Raycast som vår primære launcher og har utviklet flere extensions for å effektivisere vår arbeidsdag.

## Extensions vi har laget

### Kynd Time Tracker

En extension for å spore tid på prosjekter direkte fra Raycast.

### Kynd Project Manager

Hurtig tilgang til prosjektstatus og oppgaver.

### Kynd Slack Integration

Rask tilgang til Slack-kanaler og meldinger.

## Hvordan vi bruker Raycast

Raycast har blitt en essensiell del av vår arbeidsflyt. Vi bruker det til:

- App launching
- File search
- Clipboard management
- Custom extensions for våre interne verktøy

## Teknisk implementasjon

Våre extensions er bygget med TypeScript og følger Raycast's API-guidelines for best mulig brukeropplevelse.
