# Job Scraper Bot

En Python-basert bot som automatisk samler jobbannonser fra ulike kilder og gjør dem søkbare på ett sted.

## Funksjonalitet

- Automatisk scraping fra jobbportaler
- Databaserte søkefunksjoner
- Email-notifikasjoner for nye relevante stillinger
- Web interface for søk og filtrering

## Teknisk stack

- **Backend**: Python med FastAPI
- **Database**: PostgreSQL
- **Scraping**: BeautifulSoup og Selenium
- **Frontend**: React med TypeScript
- **Deployment**: Docker på AWS

## Hvorfor vi bygde det

Som et konsulentselskap er det viktig for oss å holde oss oppdatert på markedet. Denne boten hjelper oss med å:

- Finne relevante prosjekter
- Holde oversikt over markedstrender
- Identifisere potensielle kunder

## Fremtidige forbedringer

- Machine learning for bedre jobb-matching
- Integration med LinkedIn API
- Automatisk CV-analyse
