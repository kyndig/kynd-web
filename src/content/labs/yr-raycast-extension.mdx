# YR Raycast Extension

En Raycast extension som gir deg rask tilgang til værdata fra YR uten å åpne nettleseren.

## Funksjonalitet

- Værdata for din lokasjon
- Værmelding for de neste dagene
- Temperatur, vind og nedbør
- Åpne detaljert værdata i YR-appen

## Teknisk implementasjon

- **API**: YR API v2
- **Språk**: TypeScript
- **Raycast SDK**: v1.50+
- **Lokasjon**: Bruker Raycast's lokasjons-tjenester

## Hvorfor vi bygde det

Som konsulenter som reiser mye, er det nyttig å raskt sjekke været:

- Planlegging av reiser
- Valg av klær
- Beslutninger om transportmiddel
- Rask tilgang uten å åpne apper

## Bruksområder

- Daglig vær-sjekk
- Reisplanlegging
- Utendørs aktiviteter
- Generell værovervåkning

## Fremtidige forbedringer

- Støtte for flere lokasjoner
- Væralarmer
- Historisk værdata
- Integration med kalender
