# Kynd Design System

Et omfattende design system som sikrer konsistent brukeropplevelse på tvers av alle våre prosjekter.

## Komponenter

### Grunnleggende komponenter

- Buttons og links
- Form elements
- Typography
- Color palette
- Spacing system

### Avanserte komponenter

- Navigation
- Cards og layouts
- Data visualization
- Interactive elements

## Implementasjon

Design systemet er implementert i flere teknologier:

- **React**: Storybook med TypeScript
- **Astro**: Komponenter for denne nettsiden
- **Figma**: Design tokens og komponentbibliotek

## Fordeler

- Konsistent brukeropplevelse
- Raskere utvikling
- Enklere vedlikehold
- Bedre samarbeid mellom design og utvikling

## Bruk i prosjekter

Alle våre kundeprosjekter bruker komponenter fra design systemet som utgangspunkt, tilpasset til kundens behov og merkevare.
