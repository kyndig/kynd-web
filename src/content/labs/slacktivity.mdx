# Slacktivity

En intelligent Slack-bot som hjelper med å holde kanaler aktive og engasjerende gjennom automatiske aktiviteter og påminnelser.

## Funksjonalitet

- Daglige påminnelser om standup
- Automatiske icebreaker-spørsmål
- Aktivitetssporing per kanal
- Integrasjon med kalender for møtepåminnelser
- Moro-funksjoner som daglige memes

## Teknisk implementasjon

- **Plattform**: Slack Bolt for JavaScript
- **Database**: MongoDB
- **Hosting**: Heroku
- **API**: Slack Web API
- **Språk**: Node.js med TypeScript

## Hvorfor vi bygde det

Som et distribuert team er det viktig å holde kommunikasjonen levende:

- Øke engasjement i Slack-kanaler
- Automatisere rutineoppgaver
- Skape mer sosial interaksjon
- Holde teamet koblet sammen

## Bruksområder

- Daglige standup-påminnelser
- Icebreaker-spørsmål
- Møtepåminnelser
- Moro og sosial aktivitet
- Team-building aktiviteter

## Fremtidige forbedringer

- Machine learning for bedre timing
- Integration med andre verktøy
- Anpassbare aktivitetsprofiler
- Analytics dashboard
- Støtte for flere språk
